# ValheimSundial mod

Adds a basic sundial block to the game. Tells you what time it is and when night and day occur.


## Installation (manual)

This mod depends on BepInEx and Jotunn.
Place the .dll file into <ValheimDir>/BepInEx/plugins. That's it!
